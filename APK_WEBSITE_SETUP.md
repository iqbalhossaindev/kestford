# APK website download setup

The APK file is already included in this project at:

`Apk/KestFord.apk`

What this package does:
- Shows the download popup only for Android browser visitors
- Does not show the popup in Android WebView or installed standalone mode
- Includes a "Never pop up again" checkbox using localStorage
- Adds a footer APK download button

Live APK URL after deploy:
`/Apk/KestFord.apk`
